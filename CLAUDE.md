# Practice Path

Educational website explaining physician mortgage loans to doctors, dentists, and pharmacists.

## Project Structure

- `index.html` — main landing page
- `index-medical.html` — medical-focused variant
- `logo-preview.html` — logo preview/sandbox
- `styles.css` — main stylesheet (paired with index.html)
- `styles-medical.css` — stylesheet for medical variant

## Stack

Plain HTML + CSS, no build tools or frameworks.

## Style Conventions

- Fonts: Inter (body), Merriweather (headings/brand) via Google Fonts
- Brand colors: `#003d5c` (dark navy), `#0077b6` (blue), `#0096a0` (teal)
- Layout uses `.container` wrapper for max-width centering
